# cageloso-pos-admin
## Setup
```
# Install bun
# https://bun.sh/
curl -fsSL https://bun.sh/install | bash

# git clone
git clone https://github.com/KaguraGateway/cagelogos-pos-admin.git

# package install
bun install

# Panda Codegen
bun run prepare

# Run
bun run dev
```

## 技術スタック
- Bun 1.0.0
- Next.js 13 App Router
- ArkUI
- pandaCSS
